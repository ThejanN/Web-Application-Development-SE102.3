# AuraTech - Premium Online Gadget Store

AuraTech is a premium e-commerce web application designed for luxury tech gadgets. This project was developed as a requirement for the **SE102.3 Web Based Application Development** module at NSBM Green University.

## Features

* **User Storefront**: Browse products by category, search functionality, and a dynamic shopping cart.
* **Admin Dashboard**: Secure management of products (CRUD operations) and customer inquiries.
* **Responsive Design**: Premium glassmorphism UI/UX with smooth animations.
* **Database**: SQLite3 backend to handle all data operations.

## Tech Stack

* **Frontend**: HTML5, CSS3, JavaScript
* **Backend**: Node.js, Express.js
* **Database**: SQLite3

## Prerequisites

Ensure you have the following installed on your machine:

* [Node.js](https://nodejs.org/) (v14 or higher recommended)

## Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/ThejanN/Web-Application-Development-SE102.3.git
cd Web-Application-Development-SE102.3

```


2. **Install dependencies:**
```bash
npm install

```


3. **Run the application:**
```bash
npm start

```


4. **Access the application:**
* Open your browser and go to `http://localhost:3000`
* **Admin Panel:** `http://localhost:3000/admin.html`
* **Username:** `admin`
* **Password:** `admin123`
